# Programa Compactador

Aplicativo para compactar pastas e ou arquivo

## 📦 Estrutura do Projeto

- `compactararquivos.py` → arquivo principal (ponto de entrada)
- `janela.py` → módulo para desenhar e formatar a janela
- `metodos.py` → módulo auxiliar para execução do programa
- `verificarversao.py` → módulo para verificar a versão de lançamento

## 🚀 Instalação

### Via pacote `.deb`
- bash
- sudo dpkg -i compactararquivos.deb
- sudo apt-get install -f # corrige dependências se necessário

### Via `Pyinstaller`
- bash
- pyinstaller compactararquivos.spec

### Via `.exe`
- portátil
